# Asset Placeholders

This directory contains app assets. For a production app, replace these with actual images.

## Required Assets

- **icon.png** - App icon (1024x1024)
- **splash.png** - Splash screen image
- **adaptive-icon.png** - Android adaptive icon (1024x1024)
- **favicon.png** - Web favicon (if needed)
- **noise.png** - Optional texture overlay for organic feel

## Generating Assets

Use Expo's asset generator:

```bash
npx expo-optimize
```

Or use online tools like:

- [App Icon Generator](https://appicon.co/)
- [Splash Screen Generator](https://www.appicon.co/#app-icon)

For now, the app will work with default Expo assets.
